# security-tools-python

Small Python scripts I'm writing while learning cybersecurity.  
The goal is to practice basic automation for log parsing, simple checks, and other SOC-style tasks.

## Scripts

- `failed_login_counter.py` – Count failed login attempts in a sample auth log.
- `ip_classifier.py` (planned) – Extract IP addresses from log lines and label them as internal or external ranges.

## Notes

These scripts are for learning and practice, not production use.
